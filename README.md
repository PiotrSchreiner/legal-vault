# LegalVault

Ein sicherer, digitaler Ort für die strukturierte Verwaltung und Aufbewahrung wichtiger Dokumente und rechtlicher Unterlagen.

## Über das Projekt

LegalVault dient als zentrale Schnittstelle für Nutzer, um ihre persönlichen und rechtlichen Dokumente in einer organisierten Umgebung zu verwalten. Das Projekt fokussiert sich auf eine klare Benutzeroberfläche und eine systematische Kategorisierung von Unterlagen, um den Zugriff und die Übersicht im Alltag zu erleichtern.

## Features (geplant)

- **Strukturierter Dokumenten-Safe:** Kategorisierung nach Lebensbereichen.
- **Status-Tracking:** Übersicht über erforderliche und optionale Dokumente.
- **Modernes Interface:** Optimiert für Übersichtlichkeit und schnelle Navigation.
- **Sichere Cloud-Anbindung:** Datenverwaltung über moderne Backend-Infrastruktur.

## Tech Stack

- **Frontend:** Next.js (React)
- **Styling:** Tailwind CSS
- **Backend:** Supabase

## Starten der Entwicklung

Um das Projekt lokal zu starten, müssen die Umgebungsvariablen in einer `.env.local` konfiguriert werden.

```bash
npm install
npm run dev
```
